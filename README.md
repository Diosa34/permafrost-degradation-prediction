# Permafrost Degradation Prediction

## Постановка задачи
Прогноз деградации вечной мерзлоты в условиях изменения климата. Представляет собой задачу регрессии — 
предсказание глубины протаивания в грядущий год, основываясь на климатических условиях и истории их изменений.

X: температура воздуха, осадки, координаты, тип почвы и т.д.   
y: глубина протаивания

## Датасет

Permafrost ground temperature and active layer thickness data on the Qinghai–Tibet Plateau  
Ресурс: https://figshare.com/articles/dataset/Permafrost_ground_temperature_and_active_layer_thickness_data_on_the_Qinghai_Tibet_Plateau_2001_2020_/29206613

Этот набор данных содержит долгосрочные данные о температуре грунта и толщине активного слоя
на участках вечной мерзлоты на Цинхай-Тибетском плато (QTP) за период с 2001 по 2020 год.
Данные были собраны со станций мониторинга, расположенных в различных зонах альпийской растительности и на высотах.

Описание исходных таблиц данных.

SITE: Метаданные для всех объектов наблюдения.  
Столбцы:  
Site: Место   
Region: Регионы  
Latitude: Широта в десятичных   
Longitude: Долгота в десятичных градусах  
Altitude (m): Высота над уровнем моря в метрах  
Ecological type: Тип экологии (например, альпийская тундра, альпийский луг и т.д.)  
Vegetation cover(%): Процент растительного покрова  
VMC(%): Влажность грунта в процентах  
MAGT(oC): Среднегодовая температура грунта в вечной мерзлоте (MAGT)  
Geomorphic unit: Геоморфологическая единица (например, ледяная шапка, ледяная долина и т.д.)  
Soil type: Тип почвы (например, глина, песок и т.д.)  

ALT: Годовая толщина активного слоя (ALT) с 2001 по 2020 год для каждого участка.  
Столбцы: значения столбца Site из таблицы SITE.  
Строки: годы с 2001 по 2020 год.  

TTOP: Среднегодовая температура грунта в таблице вечной мерзлоты (верхняя граница вечной мерзлоты)  
Столбцы: значения столбца Site из таблицы SITE.  
Строки: годы с 2001 по 2020 год.  

PT10m: Температура грунта на глубине 10 метров, среднегодовые значения  
Столбцы: значения столбца Site из таблицы SITE.  
Строки: годы с 2001 по 2020 год.  

PT15m: Температура грунта на глубине 15 метров, среднегодовые значения  
Столбцы: значения столбца Site из таблицы SITE.  
Строки: годы с 2001 по 2020 год.  

Первичный анализ и предобработка:  
0. Формирование датасета из множества таблиц (SITE, ALT, TTOP, PT10m, PT15m)
2. Кодирование категориальных признаков
3. Выявление корреляций
4. Определение значимости признаков
5. Обработка пропусков
6. Выбросы и аномалии
7. Построение новых признаков

## Модель

XGBoost Regressor — позволяет эффективно моделировать нелинейные зависимости между 
климатическими факторами и глубиной протаивания.

Метрики качества.
Классичесикие метрики для регрессии:
MAE (Mean Absolute Error)
RMSE
R²

## Техническая реализация

**Источники данных**: csv или xlsx файлы, так как данные не обновляются в режиме реального времени. Чаще всего загружаются новые сведения за прошедший год.  
**Система для хранения обработанных данных**: PostgreSQL, так как требуется работать с табличными структурированными данными.  
**Предобработка данных**: Python, с использованием библиотек pandas и scikit-learn для обработки данных и построения модели.  
**Система оркестрации**: Prefect, так как она легковесная и подходит для периодических задач.  
 
## Feature engineering
1. Построение TimeSeries, лаги (t-1, t-2) для учета временной структуры.
2. Rolling mean среднегодовой / сезонной температуры
3. Координаты можно преобразовать в кластеры регионов

Альтернативные модели:
1. Линейная регрессия.
2. Random Forest.
Проще,чем XGBoost, менее чувствителен к параметрам, хуже ловит сложные зависимости.
